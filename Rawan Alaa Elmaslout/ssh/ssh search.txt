SSH (Secure Shell)
1. What is SSH?

SSH (Secure Shell) is a network protocol used to securely connect to another computer over a network. It allows users to log in remotely, transfer files, and execute commands safely.

SSH encrypts communication between computers so that passwords and data cannot be stolen by attackers.

Example:

Connecting from your laptop to a server

Managing a remote Linux machine

Using GitHub via SSH
